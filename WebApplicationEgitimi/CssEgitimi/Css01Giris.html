Css Söz Dizilimi

Css kural kümesi bir seçici ve bildirim bloğundan oluşur:

Örneğin h1 {color:blue;} şeklinde bir bildirimde bulunursak burada h1 seçici yani css'i uygulamak istediğimiz elemandır,küme parantezi içindeki color property yani özelliktir, blue ise bu özelliğe atanan değer(valıue)dir.

Satır içi (İnline)

Satır içi (inline)Css : bu elementte kullandığımız style= "background-color:brown; color:bisque;" kodları ile yazım şekline inline css denir. Bu şekilde de html elementlerine css yazabiliriz

Style kullanarak html kontrolüne yazı tipi fontu verme

Html elementlerinde border ile çerçeve kullanabiliriz

Html elementlerinde renk ismi yerine renk kodları da kullanabiliriz

Html öğresi için birden fazla stil belirtildiğinde hangi stil kullanılır?

1- Satır içi stili(İnline CSS) (html öğresi içinde h3 style=color: blue; şeklinde yazarsak)
2- Dış ve iç stil sayfaları (baş bölümde belittiğim gibi css'lerin sırasına göre)
3-Tarayıcı varsayılan (her tarayıcının html ekranınlarına uyguladığı varsayılan css özellikleri vardır)
